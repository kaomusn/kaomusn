Tanzu Kubernetes Grid Instance
The TKG instance is a complete deployment of Tanzu Kubernetes Grid (as shown in the figure above), including the management cluster, the deployed Tanzu Kubernetes cluster, and the configured shared and intra-cluster services.
Different Tanzu Kubernetes Grid instances can be deployed for different environments such as production, trial operation and testing.
Of course, as mentioned earlier, TKG instances are suitable for different IaaS providers or environments, such as vSphere private cloud, Azure and Amazon EC2; they can also be deployed in different failure domains, such as Datacenter-1, AWS us-east-2 or AWS us-west-2.

Bootstrap Machine
The boot machine is to download and run Tanzu Kubernetes Grid CLI on a portable computer, host or server, and push the management cluster to initially guide the platform to be run.
